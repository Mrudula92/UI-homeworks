what is bower? understand following commands and expalin about?
Bower is a package manager. It can manage components that contain HTML, CSS, Javascript, fonts or even image files. It fetches and installs packages.
bower init -- it creates bower.json file
bower install -- it installs packages to bower_components.
bower install package --save new dependencies to bower.json
